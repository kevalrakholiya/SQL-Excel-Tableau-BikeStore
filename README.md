# 🚴 BikeStore Data Visualization (SQL, Excel, Tableau)

A data visualization project leveraging **SQL, Excel, and Tableau** to analyze BikeStore sales data and deliver actionable insights for business decision-making.

---

## 📊 Overview

This project focuses on transforming raw sales data into meaningful insights through structured querying and interactive visualizations. The dashboard enables stakeholders to monitor performance, identify trends, and make data-driven decisions across multiple business dimensions.

---

## 🎯 Objectives

- Provide a clear view of **revenue performance**
- Enable analysis across **time, geography, and product categories**
- Support **management decision-making** with interactive dashboards
- Identify **top-performing products, regions, and sales representatives**

---

## 📈 Key Features

- **Revenue Analysis:** Track revenue trends by year and month  
- **Geographical Insights:** Analyze sales performance by state  
- **Product-Level Analysis:** Evaluate revenue by category and brand  
- **Sales Performance:** Monitor salesperson contributions  
- **Customer Insights:** Identify top customers by revenue  

---

## 🛠️ Tech Stack

- **SQL:** Data extraction, transformation, and querying  
- **Excel:** Data cleaning and preprocessing  
- **Tableau:** Interactive dashboard and visualization  

---

## 🚀 Business Impact

- Improved visibility into **sales performance and trends**  
- Enabled faster and more informed **strategic decision-making**  
- Simplified complex datasets into **intuitive visual insights**  

---

## 📌 Conclusion

This project demonstrates the ability to combine **data engineering, analysis, and visualization** to create a comprehensive BI solution that supports real-world business needs.
